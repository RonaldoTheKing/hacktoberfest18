A Page

1. How can I log in to the contest?
Enter your email id (the same one you used for the registration) and the secret code provided during the contest registration to log-in and start the contest.
2. How to get the secret code?
You can get it from Campus Commune, which is there on the right-hand side of the contest page.
3. After starting the coding problem, can I go back to the welcome page?
Yes, you can
4. Should I attempt the questions in the given order?
No, you don’t have to attempt the question in any particular order.
5. Where can I find the list of languages and compilers with their version?
You can find them in the welcome page under ‘Languages’ tab. Click on the ‘+’ sign to find the languages to be used and the compiler version.
6. If I move to question 2 without submitting question 1, does the timer continue for question 1?
Yes
7. How many questions are there in the contest round?
There will be 6 Programming Challenges.
8. If I have referred or used a code from the internet, do I have to declare the same?
Yes, you should. Before you submit the code, you will have an option (I found this code from another source). Select this option before submitting. You are allowed to refer other material, but you need to declare the source code.
9. How to check the status of the questions that I submitted?
For Private Test Cases: In My Submissions Tab. For Public Test Cases: In My Compile & Run Tab.
10. If I close my browser will the code written be saved on the system?
No, if you close your browser, you will lose your work. Ensure that you submit the code is saved in your local file system.
